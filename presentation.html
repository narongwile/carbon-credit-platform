Superadmin Workflows wired to API

Cross-tenant provisioning

  • Create / suspend organizations (live api.orgs)
  • License platforms per org — toggle entitlements (setEntitlements)
  • Provision platform wizard → creates org + licenses product
  • Provision device nodes (id, mqtt_prefix, geo)

Cross-tenant operations

  • Monitoring: live status across every org (filter org/domain/status)
  • Drill into any node twin / event log
  • Theme grants per org display
  • Audit / logs / license views

Enforcement: requireRole() = superadmin-only on org/entitlement/node-provisioning routes.

Admin Workflows (own org) wired

People & access

  • CRUD departments
  • CRUD users + roles
  • Product access per dept & per user
  • Dashboard-view permissions

Alarms & events

  • Domain-aware alarm config + apply-to-all-org
  • Event-problem catalog (root causes per dept/domain)
  • Org-wide notifications + escalation
  • Downlink: push config/cmd/OTA to devices

Reports & assets

  • Scheduled reports (CSV email, cron)
  • Org logo upload
  • Live fleet / device location map
  • Refrigeration & transformer overviews (live)

Enforcement: requireRole('admin') + org-scope — an admin can never touch another tenant.
